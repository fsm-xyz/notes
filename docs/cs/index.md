# 计算机科学

Computer Science

记录曾经折腾过的，记录下，缅怀下好动的心

大家都喜欢用高性能的库，但是让自己去写，都不愿意花很多时间

## 目的

1. 做笔记
2. 对知识划分层次
3. 学会表达自己的理解
4. 知识体系搭建
5. 实践记录

## 知识层次

一个知识点

你接触了，是一个层次

你用会了，又是一个层次

你弄懂了，那又是一个层次

你写出来让别人懂，那又是更高的一个层次

你用最通俗的言语，把知识点讲出来，让别人一下子明白，这是一个很高的层次

## 境界

高手都是信手捏来，随便就用生活中的例子，给你讲解复杂的概念，论证观点

## 创新

在日复一日重复和机械的劳动中，我们已经丧失了思考能力，没有太多自己的想法，变得四班，麻木，愚昧，毫无灵性，没有一点创新能力

现在的软件行业，类似八股取士，而不是兴趣导向

## 切入

无论做什么，都要学会找到重点，抓住核心，就跟玩游戏一样，要知道团战怎么打，整体怎么运营

## 熟悉

刚接触陌生的概念，都会觉得生硬，无法理解，随着时间的推移，慢慢就上手了

## 信息

当代社会，处处给你一种焦虑的信息，让你不知所措，垃圾消息影响你的脑力判断

看的越多越恐慌，要抓住核心，然后专注

不是谁比谁就天资高多少，而是别人舍得花时间去折腾，研究，思考，有些付出是你看不到的

## 思维

思维上的转变是非常困难的事情，但是不断的摸索，有了合适的契机，豁然开朗

量变引起质变
