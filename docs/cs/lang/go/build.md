# GO

## 构建参数

```sh
# -trimpath 移除代码路径信息
# -ldflags "-s -w" 移除符号表和调试信息
go build -trimpath -ldflags "-s -w" -o app main.go
```
## UPX 压缩

upx 压缩后的程序和压缩前的程序一样，无需解压仍然能够正常地运行，这种压缩方法称之为带壳压缩，压缩包含两个部分：

在程序开头或其他合适的地方插入解压代码；
将程序的其他部分压缩。
执行时，也包含两个部分：

首先执行的是程序开头的插入的解压代码，将原来的程序在内存中解压出来；
再执行解压后的程序。

```sh
upx -9 main
```
